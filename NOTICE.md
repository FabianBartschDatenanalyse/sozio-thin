# Notices

Sozio Thin source code is licensed under Apache-2.0.

The `catalog/` directory contains derived metadata profiles and links to public
datasets. It does not grant rights to the underlying datasets. Each profile
contains publisher attribution and a link where the current source-specific
terms must be checked.

The catalog was created from publicly available Swiss open-data metadata. No
API keys, OAuth credentials, downloaded source datasets, or private user data
are included.
