Sozio Thin

Der MCP-Server und 100 kuratierte Schweizer Open-Data-Profile sind lokal installiert.

Nach der Installation

Codex, Claude Code oder Claude Desktop vollständig beenden und neu starten. Im Client nach einem MCP-Server namens sozio-thin suchen. Das Installationsresultat steht in integration-status.txt im Installationsordner.

Testfrage

Welche Tools stellt sozio-thin bereit?
Suche danach Ressourcen zur Bevölkerung nach Alter, Geschlecht und Kanton
und zeige mir das Context Bundle.

Manuelle Codex-Konfiguration

Falls Codex beim Installieren nicht ausgewählt wurde, in %USERPROFILE%\.codex\config.toml ergänzen:

[mcp_servers.sozio-thin]
command = "C:\\Users\\DEINNAME\\AppData\\Local\\Programs\\Sozio Thin\\sozio-thin.exe"
args = ["mcp-server"]
startup_timeout_sec = 30
tool_timeout_sec = 300

Manuelle Claude-Code-Konfiguration

claude mcp add --scope user --transport stdio sozio-thin -- "$env:LOCALAPPDATA\Programs\Sozio Thin\sozio-thin.exe" mcp-server

Danach mit claude mcp list oder innerhalb von Claude Code mit /mcp prüfen.

Diagnose

& "$env:LOCALAPPDATA\Programs\Sozio Thin\sozio-thin.exe" doctor

Für die Ressourcensuche werden weder Docker noch Vespa, Gemini oder ein API-Key benötigt. Öffentliche Quelldaten werden erst bei einer Analyse von den jeweiligen Herausgebern geladen.
